## Informations
Made on Windows 7 using:
 - Python 3.7
 - pandas
 - jsonschema
 
## Usage
 - Create and activate a virtual environment using Python 3.5+
 - Install libraries
```
pip install -r requirements.txt
```
 - cd to directory
 - Add path to virtual env (should be optional)
 - Run script
 ```
 python exercise.py
 ```
 
 ## Tests
 Run tests using
 ```
 python -m unittest discover
 ```
